# Bcrypt-CLI #

<br>

Dieses Repo enhält den Quellcode für eine einfache Commandozeilen-Applikation mit *Spring Boot*,
mit der ein über Tastatur eingegebenes Passwort mit [Bcrypt](https://en.wikipedia.org/wiki/Bcrypt)
verhasht wird.
